Guia de Configuração das APIs
🔵 Google Calendar & Meu Negócio
1. Acesse console.cloud.google.com
2. Crie um projeto novo (ex: "Dashboard Pessoal")
3. Menu → APIs e Serviços → Biblioteca
4. Ative: Google Calendar API e My Business API
5. Menu → Credenciais → Criar → ID do cliente OAuth 2.0
6. Tipo: Aplicativo da Web
7. Origem JS autorizada: —
8. URI de redirecionamento: —
9. Copie Client ID e Client Secret
10. Adicione nas variáveis de ambiente do Vercel: GOOGLE_CLIENT_SECRET
🔵 Instagram & Facebook
1. Acesse developers.facebook.com
2. Meus Aplicativos → Criar Aplicativo
3. Tipo: Consumidor (ou Empresa)
4. Adicione o produto Login do Facebook
5. Em Configurações do Login → URIs de redirecionamento OAuth válidos:
    —
6. Copie o ID do Aplicativo (App ID)
7. Cole no campo Meta App ID nas configurações abaixo
8. Sua conta Instagram deve ser Business vinculada a uma Página Facebook
📄 Formato do config.json
{
  "google": {
    "clientId": "SEU_CLIENT_ID.apps.googleusercontent.com",
    "clientSecret": "SEU_CLIENT_SECRET",
    "scopes": ["https://www.googleapis.com/auth/calendar.readonly",
               "https://www.googleapis.com/auth/business.manage"]
  },
  "meta": {
    "appId": "SEU_APP_ID_AQUI"
  }
}
O arquivo config.json já existe na pasta do dashboard. Abra-o com qualquer editor de texto e substitua os valores.
